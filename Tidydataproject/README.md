# TidyDataProject
The goal of this project is to illustrate the power of data cleaning. We were able to turn what was a fairly messy, poorly organized dataset into something digestable and useable with just a few functions. 

The key is understanding some of the principles of tidy data. There should only be one column per variable and each row should be a distinct observation. There should not be more than one variable per column. Using these principles as guidelines, our file uses the "melt" and "split" functions to ensure that each variable is in one column. The project also uses the "replace" function. Please see the project for more. It is designed to illustrate how these tools can be used to make a disorderly or poor dataset manageable. 

# How to run the project

Open the "notes3" file in VSCode and simply run each box of code. The code will output the relevant data at each step. Make sure to run the cells with code and not the "markdown" cells which just have descriptions of what the is happening in the project. 

# Dataset Description
The dataset is a spreadsheet containing information on Olympic medalists in the 2008 Olympics. You can view the csv file in this folder to see how the dataset initially looks.
